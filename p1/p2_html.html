Ejemplos de Listas en HTML

Listas No Ordenadas

Las listas no ordenadas son listas que se utilizan para poder enlistar elementos que no tienen un importancia en su orden como por ejemplo una lista del supermercado.

Para definir una lista no ordenada html establece dos etiquetas una para definir donde empieza y terminar el tipo de lista y otra para definir el inicio y final de un elemento de la lista. La etiqueta ul es para definir listas no Ordenadas (unordered lists -> ul) dentro de estas etiquetas solo pueden estar definidas los elementos de listas que se declaran con la etiqueta li (list item -> li) dentro de los li pueden establecerse cualquier otro elemento html válido. El UL define que tipo de simbolo se utilizara en este caso UL es vigulillas (circulitos) Cada LI es un elemento. Un circulito.

Listas Ordenadas

Las lista ordenadas se declaran con la etiqueta ol (ordered list -> ol) En este tipo de Listas el orden es muy importante ya que demuestra un predilección por el primero o dandole mas importancia o establece un orden secuencial del mensaje a enviar.

Nuevamente, dentro de ol SOLO y UNICAMENTE pueden declarase etiquetas de elementos de lista que son exactamente igual a las anteriores li
